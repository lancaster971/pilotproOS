PilotProOS - Sistema di Gestione Documentale RAG

Questo è un documento di test per verificare il funzionamento del sistema RAG (Retrieval-Augmented Generation).

Il sistema utilizza le seguenti tecnologie:
- ChromaDB per il database vettoriale
- OpenAI text-embedding-3-large per gli embeddings (3072 dimensioni)
- Chunking ottimizzato: 600 caratteri con overlap di 250 caratteri
- Frontend Vue 3 con PrimeVue per l'interfaccia utente

Funzionalità principali:
1. Upload multiplo di documenti (PDF, DOCX, TXT, MD, HTML)
2. Ricerca semantica con query in linguaggio naturale
3. Gestione categorie e tag
4. Visualizzazione statistiche in tempo reale
5. CRUD completo sui documenti

Test di ricerca semantica:
- "Come configurare il sistema" dovrebbe trovare questo documento
- "Tecnologie utilizzate" dovrebbe mostrare la sezione tecnologie
- "Database vettoriale" dovrebbe trovare ChromaDB

Note tecniche:
- Accuracy attesa: 85-90% dopo ottimizzazioni
- Costo embeddings: $0.00013 per 1K tokens
- Supporto multilingua (italiano/inglese)

Questo documento serve per testare tutte le funzionalità del sistema RAG.
